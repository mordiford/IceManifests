# IceManifests

## About

- `local_manifests` for mordiford-ify AICP.
- ただのリンク集です。
- `vendor_aicp` と `packages_apps_AicpExtras` には独自の変更が少し入っています
    - それ以外の `patch.xml` の中身は翻訳が進んでいたりします

## AICP info

- mordiford-AICP の概要とダウンロード: https://wiki.maud.io/aicp/oreo
